Onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

Omdat bij internetadressen hoofdletters belangrijk zijn. Ook is het een gangbare naam voor de eerste pagina van een website. Het helpt ook om de pagina te vinden wat goed is voor bezoekers.

Met welke HTML versie werken we nu?

HTML5.

Met welke CSS versie werken we nu?

CSS3.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Omdat elke browser een andere rendering engine gebruikt bijvoorbeed Blink voor Chrome en Opera. Het kan ook komen omdat niet alle browsers dezelfde CSS functies hebben.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome is op dit moment de populairste browser. Gemiddeld gebruikt Chrome 500 MB tot en met 1 GB RAM.
